## File formats

- `*-vector-master.svg` - true scalable vector master. Open and edit in Adobe Illustrator, Affinity Designer, CorelDRAW, or Inkscape. Best general source file.
- `*-illustrator-editable.eps` - editable vector source for Adobe Illustrator and traditional print workflows.
- `*-print-vector.pdf` - vector PDF sized for print production and large-format output. Artwork remains scalable.
- `*-transparent-300dpi.png` - 6000-pixel transparent PNG with 300-DPI metadata for watermarking, overlays, apparel mockups, and high-resolution raster use.
- `*-online.jpg` - 3000-pixel white-background JPG optimized for websites, email attachments, and social posting.
- `mason-extensions-logo-suite-email.pdf` - four-page presentation PDF containing the full approved suite.

## Production notes

- Artwork is monochrome black and white.
- SVG, EPS, and print PDF exports are vector-based and can scale without pixelation.
- Use the SVG or EPS master when editing in Adobe Illustrator.
- Convert the pure black artwork to the print provider's requested black recipe if a specific CMYK profile is required.
- Keep clear space around each mark and do not stretch, skew, recolor, or rearrange individual elements without creating a new approved variation.
