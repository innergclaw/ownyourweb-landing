## Project
Landing page for OwnYourWeb Systems — self-hosted web infrastructure for creators and businesses who want control.

## Tech Stack
- **HTML5** — Semantic markup
- **CSS3** — Modern styling, no frameworks
- **Inter Font** — Clean typography
- **Responsive** — Mobile-first design

## How to Run

### Local Preview
```bash

# Using Python
cd /Users/ngvisions/.openclaw/workspace/projects/ownyourweb
python3 -m http.server 8000

# Using Node.js
npx serve .

# Or just open index.html directly in your browser
open index.html
```

### Deploy
Options:
1. **Static hosting** — Netlify, Vercel, GitHub Pages
2. **Self-hosted** — Nginx, Apache, Caddy
3. **Square** — SquareSpace (custom template)
